👬 Dost Lang

A Unique Toy Programming Language Built with C++ Interpreter

✨ Language Overview

Dost Lang is a custom hobby language designed to understand the core concepts of Compiler Design, Lexical Analysis, and Parsing. It reads logic directly from a .dost text file and executes it via a custom-built C++ translation engine.

💻 Code Syntax Example

bhai start 

bhai baddal = 100 ; 
bol(baddal) ; 
bol("Running_Successfully!") ; 

bhai sesh

🚀 Output Result

--- Dost Lang Output ---
100
Running_Successfully!
Project Completed successfully!
