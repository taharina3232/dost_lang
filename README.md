# 👬 Dost Lang - A Custom Toy Language Built in C++

Dost Lang is a fun, unique toy programming language inspired by Bhailang. It features a custom interpreter written entirely in C++ that tokenizes and parses code from an external `.dost` file.

## 🚀 Syntax Guide

Every program must begin with `bhai start` and terminate with `bhai sesh`.

### 1. Declaring Variables
Variables are declared using the `bhai` keyword followed by an assignment operator.
```text
bhai variable_name = value ;
